## Setup and installation

- Clone the repository
    ```
    git clone https://ghithub.com/rajdip-b/offiql-email-app
    ```
- Move into the folder
    ```
    cd offiql-email-app
    ```
- Install dependencies
   ```
    ./mvnw clean install
    ```
  
## Run JUnit tests
```
./mvnw test
```

## Run the application
```
./mvnw spring-boot:run
```
